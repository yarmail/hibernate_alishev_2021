JPA - Java Persistence Api -  спецификация Java о том,
как сохранять объекты Java в базе данных и Hibernate это
одна из библиотек для сохранения объектов.
Все библиотеки, поддерживающие JPA должны выполнять
её спецификацию

Разница между save() и persist()
В JPA нет метода save() (который есть только в Hibernate),
есть только persist() любой JPA провайдер должен иметь
спецификацию persist()

save() возвращает значение первичного
ключа для добавленной сущности. Гарантирует, что значение
первичного ключа будет определено сразу после вызова метода
